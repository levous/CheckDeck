SCLabelCell class reference

SCControlCell : SCTableViewCell : NSObject
SCTableViewCell.h

Overview

This class functions as a cell with a UILabel control. The bound property name or bound key value of this cell must be of type NSString.

The cell's bound object (see SCTableViewCell class overview).

The cell's bound property name corresponding to the UILabel's text. Property must be of type NSString and can be a readonly property.
