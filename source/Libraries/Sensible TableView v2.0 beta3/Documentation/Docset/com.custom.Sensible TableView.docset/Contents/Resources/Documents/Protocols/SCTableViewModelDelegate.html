SCTableViewModelDelegate protocol reference

SCTableViewModel.h

Overview

This protocol should be adopted by objects that want to mediate as a delegate for SCTableViewModel. All methods for this protocol are optional.

tableViewModel:itemCreatedForSectionAtIndex:item:

- (void)tableViewModel:(SCTableViewModel *) tableViewModel itemCreatedForSectionAtIndex:(NSUInteger) index item:(NSObject *) item
Parameters
tableViewModel

The model informing the delegate of the event.

index

The index of the section who's a new item has been created for.

item

The item that has been created.

Discussion

Notifies the delegate that a new item bound cell has been created for the section at the specified index. Note: Method usually called for SCArrayOfItemsSection's subclasses. Important: Although item has been created, it's still not added to the section and can be deallocated if the user cancels the item's detail view.
