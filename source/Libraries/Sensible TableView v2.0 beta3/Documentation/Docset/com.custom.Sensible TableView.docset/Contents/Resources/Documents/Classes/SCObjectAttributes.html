SCObjectAttributes class reference

SCPropertyAttributes : NSObject
SCPropertyAttributes.h

Overview

This class is used to extend the definition of an SCPropertyAttributes instance of type SCPropertyTypeObject, thus allowing further customization of the generated control by the user.

Properties

classDefinitions

@property NSMutableDictionary *classDefinitions
Discussion

Should contain all the different class definitions needed by the generated object control. Each dictionary entry should contain a key with the SCClassDefinition class name, and a value with the actual SCClassDefinition.

Tip: The class name of the SCClassDefinition can be easily determined using the SClassDefinition.className property.
